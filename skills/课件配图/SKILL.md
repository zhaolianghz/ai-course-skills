# 课件配图

把课件里的一个**认知锚点**(判断/流程/结构/状态/隐喻)变成一张有记忆点的示意图——
不是配装饰图。借鉴 ian-xiaohei-illustrations 的配图方法论,皮换成公司课件风。

> 核心信念:**配图 = 把一个关键认知动作画出来**,不是"凑张图让页面不空"。
> 一张图只讲一个核心结构;装饰性插图(无关照片/泛泛插画)= 反面教材,不做。

## ⛔ 三条铁律

1. **一图一锚点**:每张图只表达一个判断/流程/结构/状态/隐喻,不堆砌。
2. **教学化、和课件同源**:风格服从 cobalt-grid + Hyreal OPClub logo,不用怪诞手绘/卡通/IP。
   (这点和小黑相反——小黑刻意"非教学化",我们刻意"教学清晰"。)
3. **按内容定图,不教条**:这一页讲什么,就配最合适的那种图——**不是所有课都画 SVG**。
   工具操作步骤→**真实操作截图**;抽象概念/结构→示意图(SVG);数据/对比→图表;真实成果→产出截图;
   流程→流程图;需手绘风→插画(图像API)。先判断"该用哪种图",再选做法。
4. **不溢出固定舞台**:幻灯片是固定 1920×1080,**一页最多一个重元素**(代码块 OR 图 OR 表格,三选一)。
   标题+大代码块+图+多段文字塞一页必溢出、被切掉、和翻页控件叠。**超了就拆页,不要压缩塞进去**
   (frontend-slides 铁律:"内容超出密度就拆成更多页,不要缩到挤")。图加 `max-height:620px` 兜底。

## 工作流(5 步)

### Step 1 出 shot list(先规划,不急着画)
通读课件/文章,提炼**值得画的认知锚点**,先列清单——每张图写:放在哪页/哪段后、主题(一句话)、
核心意思(让人记住什么)、结构类型(8 选 1)、建议元素 + 标注词(≤5 个短词)。

**不是每页都配图**。只在"光文字说不清的结构/流程/对比/隐喻"处配。

### Step 2 选结构类型(8 选 1,先选再画)
按"想让学员记住一个什么**动作**"选 → 见 [references/structure-types.md](references/structure-types.md)
(8 类结构表 + 进阶负空间构图 + "图的种类与做法"对照表)。

### Step 3 出图(按种类选做法)
- **抽象概念/结构/流程** → 手画 cobalt 风 SVG。画法(配色/留白/标注/marker)+ 插入方式见
  [references/svg-style.md](references/svg-style.md)。
- **工具操作步骤/成果** → 真实操作截图(浏览器自动化+人工登录,**不可降级为占位**)。
- **配不出真图时** → 走三级降级,**永不留白板**,见
  [references/fallback.md](references/fallback.md):
  L1 结构占位卡 / L2 提示词占位,均用 `scripts/fallback_card.py` 一行生成:
  ```bash
  python3 scripts/fallback_card.py --type workflow --title "取材→写稿→验收" --out card.svg
  python3 scripts/fallback_card.py --list   # 看支持的结构类型
  ```

### Step 4 QA 自检 + 截图验证(生成后必做)
逐项 QA + headless 截图核对(别只肉眼猜,溢出/贴错页看不出来)——清单与命令见
[references/svg-style.md](references/svg-style.md) 末两节。

### Step 5 收尾
报告:配了几张、各放哪页、哪些是占位卡/提示词占位(待替换)、有无未完成截图位。

## 参考来源
方法论借鉴 [ian-xiaohei-illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations)
的**结构类型库 + shot list + QA**;降级链借鉴 xhs-publisher(配图失败兜底模板)+
content-agent(无 Key 输出提示词占位)。**视觉风格相反**:它们怪诞非教学,我们 cobalt 教学化。借方法不借皮。

## 不做什么
- 不配装饰图/无关照片/泛泛插画(铁律 1)
- 不在一张图里塞多个概念;不在图里写标题(交给幻灯片标题)
- 不引入怪诞 IP/卡通风(和企业课件调性冲突)
- 不每页都硬配图(只在文字说不清处配)
- 工具步骤**不允许**用占位代替真截图(占位=课没做完,发布门禁会拦)
