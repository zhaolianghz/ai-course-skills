# 卡点格式

## 卡点 1：选题定位确认

在 Step 1 后停止，向用户输出：

```markdown
## 卡点 1：选题定位确认
- KP 编码：
- 课程分类：L 理论 / P 实践 / C 案例 / T 工具 / R 综合项目
- 课程名：
- 适学对象：
- 先修：
- 本课展开：
- 只给指针、不展开：

请确认是否进入取材和大纲阶段。回复“确认”继续，或指出要改哪一项。
```

用户说“改 X”时，只改 X 并重新确认，不要重跑全流程。

## 卡点 2：大纲结构确认

在正文填充前停止，向用户输出：

```markdown
## 卡点 2：大纲结构确认
- 课程名：
- 一句话定位：
- 章节结构：
- 课时/阶段配比：
- 每章要解决的问题：
- 明确不展开的相邻 KP：

请确认是否填充正文。回复“确认”继续，或指出要改哪一章。
```

确认后再写正文。正文完成后的局部修改优先定点编辑，不要整体重生成。
