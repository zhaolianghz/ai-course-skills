# 焕燃科技 · AI Coding Skills

公司课程研发用的 AI Coding 技能集（Claude Code / Codex / WorkBuddy 通用）。

## Skills

### 课件母版创建
按课程研发 SOP 自动创建一门课的「课件内容母版」。支持产出 Markdown 母版 + SVG 配图 + HTML 幻灯片 + PPTX 课件，并推送到飞书知识库「焕燃课程 → 课程待审区」等待审核。

触发：「做 KP-X.X 的课件」「创建课件母版」「写某课程的课件」。

流程：选题定位 → 取材 → 大纲骨架 → 写母版 → 自检 → 登记索引 → 生成 HTML/PPTX → 推送飞书（三处卡点：选题定位后、大纲拆分后、推送飞书前）。

### 课件配图
给课件/课程决定配什么图、怎么配。把抽象认知点（流程/结构/状态/隐喻）变成示意图，而非装饰图。

触发：「给这页/这门课加图」「配图」「这页太干」。

含：8 种结构类型库 + shot list 先行 + cobalt 风 SVG 规范 + QA 自检。

### 视频生成
用 AI 视频平台（即梦 / Seedance 等）真机生成 AI 视频，产出可用样片，用于课程实拍位填充或演示素材。

触发：「生成视频」「生个样片」「做条短视频」「用 Seedance 生成」。

### 课程种草视频
把一门课的真实大纲变成一支会卖课的种草短视频（竖屏，可直接发抖音/小红书/视频号）。

触发：「给 X 课做种草视频」「做条宣传片」「课程种草」「宣发视频」。

### wechat-article-capture
抓取微信公众号文章（含图片），洗稿后保存到 IMA 知识库笔记中。

触发：分享微信公众号文章链接 + 要求保存到知识库。

## 安装

| 平台 | 路径 |
|------|------|
| WorkBuddy | `~/.workbuddy/skills/` |
| Claude Code | `.claude/skills/` |
| Codex | `.codex/skills/`（或按 Codex 约定） |

放到对应目录下即可被自动发现加载。

## 依赖

- **飞书连接器**（feishu connector）：课件母版创建需要，用于推送到飞书知识库
- **IMA OpenAPI**：wechat-article-capture 需要，用于保存笔记到知识库
